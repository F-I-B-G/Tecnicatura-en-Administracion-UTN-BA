# TP 1 (Excel)

### Planilla de Gestión de Recursos Humanos y Liquidación

Análisis de procesos de liquidación, aplicación de fórmulas complejas, modelado de datos y propuestas de mejora para la optimización administrativa.

# TP 2 (Benchmarking)

### Benchmarking de Software GTD - Análisis de Sistemas de Productividad

Aplicación de metodología de ponderación para la evaluación técnica de herramientas. Criterios de compatibilidad, seguridad y ROI para la toma de decisiones tecnológicas.

# TP 3 (Access DB)

### Diseño y Modelado de Base de Datos para Gestión Empresarial (Sun Gym)

Modelado Conceptual y Físico de una Base de Datos Relacional (DER). Implementación de integridad referencial (claves primarias/foráneas) para asegurar la calidad del dato. Diseño orientado a la escalabilidad y soporte para Consultas SQL (BI/Reportes).

# TP 4 (Cursograma)

### Análisis y Documentación de Procesos de Negocio (Cursograma)

Modelado de flujo de información para el proceso de ventas. Desarrollo de documentación clave para la futura digitalización y análisis de datos (BI) del negocio.

# TP 5 (ERP - Módulos de Finanzas y Tesorería)

### Análisis de Sistemas ERP: Módulos de Finanzas y Tesorería

Evaluación de módulos clave (Finanzas y Tesorería) de un sistema ERP. Modelado del flujo de información integrado y la integración entre procesos (Contabilidad/Tesorería). Demostración de cómo un ERP centraliza datos para la Toma de Decisiones Estratégicas y el control interno.

# TP 6 (Colppy)

### Implementación Práctica de Procesos Contables en ERP (Colppy)

Ejecución de ciclos contables completos (Facturación, Cobros, Pagos, Anulación) dentro de un software ERP. Demostración de dominio en procesos transaccionales, gestión de la cuenta corriente de clientes y proveedores, y aseguramiento de la trazabilidad de documentos (Nota de Crédito).

# TP 7 (CRM)

### Gestión de Relación con Clientes (CRM) y Automatización de Procesos (Bitrix24)

Implementación y configuración de un sistema CRM para la trazabilidad del ciclo de ventas y el servicio al cliente. Uso de herramientas de automatización y comunicaciones internas. Demostración de cómo la gestión digital de relaciones centraliza la información para impulsar el crecimiento y el Liderazgo Estratégico (Puente a BI).
